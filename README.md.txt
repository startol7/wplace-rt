# TePlace — Real World Map Conquest

TePlace は、実世界の地図（Leaflet + OSM タイル）上に、超小さなタイル（既定 `0.0001°`）を塗って陣取りするシンプルなブラウザゲームです。  
単色 / パレット / レインボーの 3 種のブラシ、ランキング・戦闘ログ・簡易ヒートマップ・グリッド表示などを備えています。

> ⚠️ オンライン人数はデモ用の**ダミー値**です。実人数をカウントしたい場合はサーバー（WebSocket 等）を追加してください。

---

## デモ / スクリーンショット
- デモ: Render（静的サイト）にデプロイ可能
- 主要 UI: マップ・ランキング・戦闘ログ・統計・ブラシ／カラーパレット
