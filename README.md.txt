# wwplace — Real World Map Conquest

- Ultra-fine grid: **~11m squares (0.0001°)**  
- Each tile belongs to a country.  
- Countries can capture each other’s tiles.  
- **Occupation Stats** panel shows which country captured how many tiles from others.  
- Colors are auto-generated (HSL) so every country gets a unique color.

## Controls
- Click the map: capture a tile (5s cooldown).  
- Grid / Heatmap toggle buttons.  
- Sidebar shows rankings, logs, and occupation stats.
